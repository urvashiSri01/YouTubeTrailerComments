# YouTubeTrailerComments
A text file with various comments by the YouTube users is provivded. The requirement is to get the various Movie Ids and information regarding the Viewership counts, likes and dislikes, etc. 

Sample desired output:
1,128238,170,7,0,7
2,155561,135,14,0,74
3,2532,7,2,0,1
4,6827,9,1,0,0
5,330807,374,16,0,121

Optional headers:
id,vc,lc,dc,fc,cc

id: MovieID
vc: View count
lc: Likes count
dc: Dislikes count
fc: Forward/Share count
cc: Comments count
